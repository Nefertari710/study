1. 连接数据库
	- mysql -u root -p
	- 需要输入密码![](../图片资源/终端数据库相关操作图片资源/P1.png)
2. 查看所有数据库
	- show databases![](../图片资源/终端数据库相关操作图片资源/P2.png)
3. 切换到shixi数据库
	- use shixi![](../图片资源/终端数据库相关操作图片资源/P3.png)
4.  查看表格
	- show tables;![](../图片资源/终端数据库相关操作图片资源/P4.png)
5. 查看userinfo中的信息
	- select * from userinfo;![](../图片资源/终端数据库相关操作图片资源/P5.png)
